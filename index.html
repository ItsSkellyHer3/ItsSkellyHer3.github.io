ChatIfy

Main Project

A private, open-source chat app. Built to keep your messages completely safe and hidden.

Why I built this
Privacy matters. I built this so people can chat without worrying about their data being collected. I used TypeScript to keep the code organized and SQL to structure and save the messages safely.

Tools I Use

React Tailwind CSS Node.js TypeScript
How I use them
I use React and Tailwind to quickly build clean websites that work on computers and phones. I write everything in TypeScript so I make fewer mistakes before the code even runs.

How I Host

Ngrok Tunneling
How it works
Instead of paying for web servers, I run my code directly from my own Linux machine. Ngrok creates a secure path straight to my computer. It is free and very simple.

Data & System

SQL MongoDB Linux Git
Why these?
I use Linux to run everything smoothly and Git to save my code updates. I choose between SQL or MongoDB depending on the type of data the project needs.
